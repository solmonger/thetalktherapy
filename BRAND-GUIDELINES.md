# The Talk Therapy - Brand Guidelines

## Overview
A warm, inviting, safe space for mental health content. The brand should feel like a comforting conversation with a trusted friend.

## Color Palette

### Primary Colors
- **Soft Peach:** `#FFD8C8` - Warm, inviting, gentle
- **Soft Sage:** `#C8D8C8` - Calming, natural, peaceful  
- **Cream:** `#FFF8F0` - Soft background, warmth
- **Warm Gray:** `#E8E0D8` - Neutral, supportive

### Accent Colors
- **Dusty Rose:** `#D8A8A8` - Compassionate, gentle
- **Muted Teal:** `#88B8B8` - Clarity, calm
- **Soft Brown:** `#B8A088` - Grounded, earthy

### Usage Guidelines
- Use Soft Peach for primary CTAs and highlights
- Soft Sage for backgrounds and section dividers
- Cream as the main background color
- Warm Gray for borders and subtle elements

## Typography

### Primary Font: Inter
- Clean, readable, modern sans-serif
- Open-source (Google Fonts)
- **Headings:** Inter SemiBold 600
- **Body:** Inter Regular 400
- **Accent:** Inter Light 300 for subtle text

### Font Sizes
- **H1:** 2.5rem (40px) - Page titles
- **H2:** 2rem (32px) - Section headers
- **H3:** 1.5rem (24px) - Sub-headers
- **Body:** 1rem (16px) - Main content
- **Small:** 0.875rem (14px) - Captions, metadata

## Spacing System

### Base Unit: 8px
All spacing multiples of 8px for consistency:
- **XS:** 4px (0.5×)
- **S:** 8px (1×)
- **M:** 16px (2×)
- **L:** 24px (3×)
- **XL:** 32px (4×)
- **XXL:** 48px (6×)

## Layout Principles

### 1. Generous Whitespace
- Let content breathe
- Minimum 24px between sections
- 48px padding on page edges

### 2. Calm Composition
- Avoid clutter
- Every element earns its place
- Balanced visual weight

### 3. Mobile-First
- Responsive from 320px up
- Stack vertically on mobile
- Maintain readability at all sizes

## Imagery Style

### Photography
- Natural, soft lighting
- Warm tones
- Authentic, not staged
- Nature elements when possible
- Avoid clinical/sterile imagery

### Illustrations
- Simple, line-based
- Organic shapes
- Soft edges, no sharp corners
- Use brand colors

## Website Design Concepts

### Concept A: "Calm Sanctuary"
- Full-width hero with soft gradient
- Card-based content sections
- Rounded corners throughout
- Subtle animations on hover

### Concept B: "Conversation Flow"  
- Asymmetrical layout
- Organic, flowing shapes
- Dialogue-inspired elements
- Warm, conversational tone

### Concept C: "Minimal Comfort"
- Ultra-clean layout
- Maximum whitespace
- Focus on typography
- Subtle texture backgrounds
